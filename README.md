### 🚀 Featured ML Projects

#### 📦 [Demand Forecasting Engine](https://github.com/shenaltissera/Demand-forecasting-engine)
Probabilistic time-series forecasting across thousands of retail SKUs (Zara-inspired). Ensembles **LightGBM** for high-volume items with **Prophet** for seasonal/slow movers, tying inventory decisions to confidence intervals — **cut simulated stockouts by 31%**. Orchestrated with **Apache Airflow**; evaluated on MAPE, stockout rate, and CI coverage.
`Python` · `LightGBM` · `Prophet` · `Airflow` · `Jupyter`

#### 🚦 [RiskRadar](https://github.com/shenaltissera/RiskRadar)
Real-time traffic-accident risk predictor that fuses **live weather data** (Open-Meteo) with an **XGBoost** classifier to forecast road-risk levels by location and time. Served via **AWS Lambda + API Gateway** with an interactive **Streamlit + PyDeck** dashboard.
`Python` · `XGBoost` · `scikit-learn` · `AWS Lambda` · `Streamlit`

#### 💹 [Finance Prediction](https://github.com/shenaltissera/Financeprediction)
End-to-end financial prediction app with a data pipeline, ML predictive modeling, and a **backtesting** engine to validate strategies before deployment — wrapped in an interactive **Streamlit** UI.
`Python` · `scikit-learn` · `Streamlit` · `pandas`
